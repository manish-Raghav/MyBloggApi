# Blog Application

A full-featured Blog Application built using Node.js, Express.js, and MongoDB.

## Features

- User Authentication
- Create Blog Posts
- Update Blog Posts
- Delete Blog Posts
- View All Blogs
- JWT Authentication
- Secure Password Hashing
- REST API
- MongoDB Database
- Error Handling
- Middleware Support

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- dotenv
